# Roadmap

This roadmap outlines the planned development steps for the RAG Knowledge Agent portfolio project.

## Phase 1: Project Setup

- Create GitHub repository
- Add README with project overview
- Add architecture documentation
- Add sample documents for testing

## Phase 2: Local .NET API

- Create a .NET Web API project
- Add an endpoint for asking questions
- Define request and response models
- Add basic validation and error handling

## Phase 3: RAG Integration

- Connect to Azure AI Search
- Retrieve relevant document chunks
- Connect to Azure OpenAI / Azure AI Foundry
- Generate grounded answers using retrieved context

## Phase 4: Source References

- Return source document names with answers
- Add confidence or relevance information
- Handle cases where no relevant source is found

## Phase 5: Polish

- Add screenshots
- Add API examples
- Add setup instructions
- Add architecture diagram
- Add demo video or GIF
